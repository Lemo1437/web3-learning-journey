# Web3 Learning Journey 🚀

## 目标
学习 Web3 开发，掌握 Solidity、智能合约、区块链基础，并结合 Vue3 进行前端开发。

## 计划
### 第一阶段：基础学习（1-2 周）
- 区块链 & Web3 基础概念
- 以太坊、智能合约、钱包（Metamask）
- Solidity 语言基础
- 使用 Hardhat 部署智能合约

### 第二阶段：前端集成（2-3 周）
- Web3.js / ethers.js 基础
- 使用 Vue3 + ethers.js 连接智能合约
- Dapp 开发基础

### 第三阶段：实战项目（3-4 周）
- 开发一个小型 Dapp（如投票系统、NFT minting 页面）
- 优化 & 安全性
- 部署到 Testnet/Mainnet
